# Data Structure - Stack

## Stack
- Linear Data Structure
- LIFO(Last In First Out)

## Stack functions
- Push : Add an item in the stack
- Pop  : Remove an item from the stack
- Top  : return top element of stack
- isEmpty : return true if stack is empty

## Implementation
- Array
- Linked list
- vector
